    前台功能¶

    Discuz! X3.4 前台使用

    论坛¶

    1. 论坛首页¶

    2. 论坛板块¶

    1. 首页居中位置
    2. 管理员可以在后台添加、并设置分区的样式。分区下的版块默认竖排显示，可在后台设置为横排显示。
    3. 可以显示主题数、帖数、新主题数、最新主题与其发布时间和作者等内容。
    4. 版块的图标、名称、简介等在后台 论坛=>版块管理=>版块编辑 下都可以个性化设置。

    3. 论坛热点¶

    1. 聚合显示热点主题
    2. 可打开或关闭，并设置显示数量：后台 界面->界面设置->论坛首页

    4. 在线会员¶

    1. 显示当前在线的会员分类和统计
    2. 在后台 界面->界面设置->论坛首页进行设置

    5. 扩展元素¶

    可DIY在此区域来展示一些吸引用户或者需要推广的内容

    门户¶

    1. 首页¶

    1. 门户类似于cms，拥有频道栏目、专题、文章投稿、审核体系、编辑人员权限分配等基础功能。
    2. 站长可以用“门户”制作一个简单的社区首页，也可以制作成类似于门户网站那样丰富的首页。
    3. 在门户应用下，包括**资讯、专题与频道**三大版块

    2. 频道¶

    1. 门户频道的主要功能是聚合社区精华，把精彩的内容聚合显示在一个页面上展示给用户看，从而吸引用户。
    2. 频道栏目，需要管理员在后台创建，点击后台 => 门户 => 频道栏目 => 添加频道

    3. 专题¶

    1. 是服务于集中报道和深度报道的一种展现形式，能集中体现某项热点事件，也可以应用到商家服务。
    2. 需要在后台 门户->专题管理 下创建，点击创建专题，进入创建专题页面：

    4. 资讯¶

    1. 资讯是指用户及时获得的有价值的信息，包括帖子、日志，也可以是社区管理人员发布的新闻。
    2. 系统提供了评论、表态动作以及邀请等元素增强互动。
    3. 提供给了发布文章、添加相关文章、模块推送、编辑、删除等相关操作来方便站长对资讯的管理。

    家园¶

    1. 家园完全实现了SNS的功能。在家园中，用户通过好友、感兴趣的话题等多种因素进行网络社交拓展，并形成放射性网络。
    2. 作为**社交网络**的每个节点，用户采用多种形式不断扩充丰富的数据，包括发表日志，上传相册、图片等。

    1. 个人空间¶

    1. 个人空间类似于QQ空间，用户可以在此发表日志、上传图片、发布记录等。

    2. 动态¶

    好友动态页面核心元素是展示好友动态以及发表记录，同时，页面左侧提供了家园的功能导航，右侧提供了任务、道具等运营元素的展示。

    广播¶

    1. 系统吸收微博优点为论坛所用，新增广播、关注、@好友、评论回流等功能。

    1. 发布广播¶

    广播类似于微博的发表，用户可以非常方便地分享自己身边一切新鲜事儿。就像发微博一样，直接在发布框输入内容，选择要@的好友，或是直接发布。

    2. 收听¶

    可以让用户收听自己感兴趣的人，了解收听人的动态。

    3. 特别收听¶

    系统还提供了“特别收听”，可以将重量级人物加入到“特别收听”列表，在页面可以直接筛选查看到他们的广播

    4. 广播大厅¶

    在广播大厅，可以看到所有人发布的广播，按时间倒序排列；可以对感兴趣的广播进行转播和回复

    群组¶

    1. 群组功能可以把有共同话题的人聚集在一起，站长可以利用群组功能打造许多细分、垂直的群组，满足不同用户需求，像车友会、影迷会等。
    2. 站长可以根据自己社区的运营情况，先设计好群组要开设哪些分类、等级，在后台做好设置。

    1. 群组首页¶

    群组首页作为群组的入口，聚合了群组的焦点，展示推荐群组、群组分类，群组的积分排行以及创建群组按钮。

    用户¶

    用户是社区的最基本元素，系统提供了灵活成熟的用户组体系、好友体系、积分体系来增强用户茉誉感、归属感，调动用户积极性，辅助站长运营。

    1. 注册、登录、找回密码¶

    系统提供了两种注册方式，即普通注册和邀请注册。
    1. 普通注册：正确填写用户名、密码、邮箱等各项内容， 验证通过，提交即可；
    2. 邀请注册：若系统只开启了邀请注册，则邀请码为必填项，用户注册时必须持有邀请码才能注册，邀请码可以通过论坛中的已注册用户获取或是在线购买。
    3. 用户可以选择使用用户名、UID、email进行登录。
    4. 如果忘记密码，可点击找回密码，填写注册时使用的Email 提交即可，系统将会将重置密码的链接发送到邮箱，点击重置即可（三天内有效）。

    2. 好友¶

    好友是将社区中的用户联系起来进行社交拓展的重要元素，用户可以通过“查找好友”结交志趣相投的好友，“可能认识的人”又为您提供和周围用户巧遇的缘分。加为好友后，我们可以打招呼、踩日志、写评论、留言、发短消息等和好友互动。
    - 加为好友：用户可以在个人空间、动态页面等地的好友名片添加用户为好友：
    - 好友请求：当有用户请求加您为好友时，系统将会发送提醒。
    - 好友分组：系统提供了好友分组功能，来方便用户对好友的管理。
    - 好友列表：系统在“好友列表”，提供了全部好友列表、在线成员、就在我附近的朋友、我的访客和我的足迹，方便用户分类查看好友。
    - 可能认识的人
    - 邀请好友。

    3. 设置¶

    系统提供了用户资料的设置功能，在此用户可以修改头像、个人资料、隐私、密码等项。

    4. 通知¶

    消息功能类似于微博的私信，方便用户间私密的交流。

    5. 积分¶

    积分系统构成了社区的价值体系，在站点中的两大作用就是衡量用户级别和参与站内交易。

    6. 用户组¶

    用户组系统构成了社区的用户体系，在社区中的作用是明确用户级别和控制用户权限。

    手机论坛¶

    为了帮助站长适应移动互联网发展趋势，Discuz! 推出了手机客户端"掌上论坛"。

    1. 将客户端下载安装到手机，首次运行客户端，页面将显示热门论坛的列表。

    2. Web版： 当论坛开启了手机版时，用手机访问论坛将自动跳转到手机版界面，手机版分为标准版和精简版。标准版为全功能手机版式；精简版为去掉小图标、主题列表日期等页面元素的精简版式；两种模式下表情均解析，用户可以根据自己需要切换在标准版、精简版和电脑版之间自由切换。
